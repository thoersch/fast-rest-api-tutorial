## Description

This is the resulting code base for the blog post regarding standing up a fully functioning RESTful API, complete with object persistence.

## Running the project

With Gradle:

```
gradle run
```

With Maven:

```
mvn package
```

then:

```
java -jar target/fast-rest-api-0.1.0.jar
```
